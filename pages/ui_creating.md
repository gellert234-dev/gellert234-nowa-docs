## Creating a screen
To create a new screen, click on the `Screen` tool from the drawing panel on the left side, then click on an an empty place on the baord to create a screen. The size of the created screen will be by default the size of `Pixel 3a`.

What if you want another screen size? it's simple. When selecting the screen tool and before you clicking on an empty place on the board, you will see multiple screen sizes inside the `details panel` on the right size. Select the size you want then click on an empty place to create the screen.
