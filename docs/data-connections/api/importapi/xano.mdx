# Xano🔌

Xano 🐉 is a **no-code backend** development platform 💻 that lets you build and scale APIs 🌐 and databases 📊 without needing to write server-side code ⌨️.  
It’s often described as a “backend-as-a-service” (BaaS) ☁️ because it provides the infrastructure 🏗️ and tools 🧰 needed to power applications 📱, mobile apps 📲, and websites 🌍.

✨ To import from Xano:

1️⃣ Get your **Xano API token** 🔑 from your Xano dashboard 📋  
2️⃣ Click on the **Import** button ⬇️ in your collection 📂  
3️⃣ Select **"Import from Xano"** 🧲  
4️⃣ Paste your Xano API token 📌  
5️⃣ Click **Import** 🎯✅  

---

🔒 **How to get your Xano API token:**  

1. Go to your **Xano dashboard** 🖥️  
2. Click on the ⚙️ **settings** for your instance 🏗️  
3. Select **"Metadata API & MCP Server"** 📑  
4. Click on **"Manage Access Token"** 🧾  
5. Click **"Create New Token"** ➕  
6. Copy 📋 the generated token 🪙  
7. ⚠️ Make sure to **save it somewhere safe** 🛡️ as you won't be able to see it again 🕵️
